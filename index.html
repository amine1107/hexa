Download Hexa

"NPM"

$ npm install hexa-animation

Or Bower

$ bower install hexa-animation

Basic Usage

you can simply adding Hexa to your html file.


            <link rel="stylesheet" type="text/css" href="hexa.css">

Adding Class

you can simply adding Hexa to your html file.


            <div class="hexa jump">

Iteration Count

hexa allows you to control the number of cycles used to repeat an animation before it stops.

Primary Class Name Type Of Animation Iteration
hexa jump x2

there are four types of iteration, and each has a number of cycles to repeat an animation :

  • x2 - two times
  • x3 - three times
  • x4 - four times
  • noStop - infinite
